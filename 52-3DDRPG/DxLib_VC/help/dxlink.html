開発支援関係をされている方のリンク


DxLib Prominence2D

　藤宮翔流様が作成されたゲームエフェクト作成ツール『Prominence』で作成したエフェクトデータをＤＸライブラリを使用したソフトで使用するためのライブラリ『DxLib Prominence2D』( ぱおらん様作 )のウェブサイトです。 『DxLib Prominence2D』を使用するとエフェクトデータの読み込みに１行、描画に１行 ( ＋アニメーション処理 )のたった数行でエフェクトを描画することができます。




　個人ゲーム開発者向けのゲームエフェクト作成ツール『Prominence3D』の開発・販売をされている
藤宮翔流様のウェブサイトです。こちらのページ でＤＸライブラリを使用したソフトで『Prominence3D』で作成したエフェクトを再生する方法を解説していただいています。


DxLibEx

　Nameless様が立ち上げられたＤＸライブラリC++化プロジェクト『DxLibEx』の GitHub のページです。


CCDxLib72

　CHY72様が開発された Cocos2d-x で動くＤＸライブラリのラッパーライブラリ「CCDxLib72」の GitHub のページです。「CCDxLib72」を使用すると Android や Mac などで動作するソフトをＤＸライブラリの関数を使用して開発することができます。


nnmy

　lriki様のウェブサイトです。ＤＸライブラリ用数学ライブラリLuminoMath 1.0.0 DxLibの他、ゲーム製作に便利なライブラリを開発されています。


DXライブラリPortable

　憂煉様のウェブサイトです。ＤＸライブラリをＰＳＰで利用することを目指したライブラリ「DXライブラリPortable」を開発されています。

　当サイトへのリンクについて

　ＤＸライブラリ置き場はリンクフリーです、どこにでも適当に遠慮なくリンク貼って良いです。
(今のところリンクを貼られて迷惑なことはこれっぽっちもありませんので)
　あと当サイトのバナーは大きいサイズと小さいサイズの２つを用意しています、味も素っ気もありませんが よかったら使ってください。





　リンクのＵＲＬは http://dxlib.o.oo7.jp/ でお願いします。


　相互リンクについて

　リンクは基本的に当サイトに来て頂いているの方が利用するものなので、プログラムやゲーム制作当の 知識や技術の向上に役に立つようなコンテンツを掲載しているＨＰを優先的に増やしていきたいと思っています。
